Identity file

In your file Explorer, go into the folder Android/data/com.kiloo.subwaysurf/ which is your game directory.

From there you find the folder /auth/subway-prod/ which contains the identity file.

Move the file to a more accessible location, such as Downloads so you can easily access it with the file picker.
